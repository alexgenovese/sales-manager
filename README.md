# Sales Manager AI Agent

A comprehensive AI sales intelligence and outreach system designed for Claude Code, following the skills.sh / Agent Skills spec.

## Overview

The Sales Manager is a high-performance agent that helps founders, sales teams, and agency owners research prospects, qualify leads, and generate personalized outreach.

## Installation

Follow the instructions in the `skills/` directory to integrate these skills into your AI agent.

## Available Skills

- `/sales prospect`: Full prospect audit.
- `/sales quick`: 60-second snapshot.
- `/sales research`: Company research.
- `/sales qualify`: Lead qualification.
- `/sales contacts`: Decision maker identification.
- `/sales outreach`: Cold outreach email sequence.
- `/sales followup`: Follow-up email sequence.
- `/sales prep`: Meeting preparation brief.
- `/sales proposal`: Client proposal generator.
- `/sales objections`: Objection handling playbook.
- `/sales icp`: Ideal Customer Profile builder.
- `/sales competitors`: Competitive intelligence.
- `/sales report`: Sales pipeline report.
- `/sales report-pdf`: Sales pipeline report (PDF).
